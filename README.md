# 🛡️ CyberShield AI

An AI-powered Cyber Resilience Platform that helps detect, analyze, and respond to cyber threats in real time.

---

## 🚀 Live Demo

🔗 https://cybershield-ai-five.vercel.app

---

## 📌 Problem Statement

Critical infrastructure such as hospitals, government systems, and enterprises face increasingly sophisticated cyber attacks.

CyberShield AI provides an intelligent dashboard that:

- Detects threats
- Analyzes attack patterns using AI
- Predicts future risks
- Recommends mitigation strategies
- Visualizes incidents in real time

---

# ✨ Features

## 🤖 AI Threat Analysis
- AI-powered threat detection
- Executive incident summary
- AI-generated response recommendations
- Threat prediction
- Confidence scoring

## 🛡️ Cyber Attack Simulations

- Malware Attack
- Ransomware Attack
- DDoS Attack
- Insider Threat
- Phishing Attack

Each simulation dynamically updates the dashboard with AI-generated insights.

## 🧠 AI Capabilities

CyberShield AI uses Google's Gemini API to:

- Analyze security logs
- Classify threats
- Generate incident summaries
- Predict future attack risks
- Recommend response actions

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/anzel27/cybershield-ai.git
```

Install dependencies

```bash
npm install
```

Create a `.env.local`

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the development server

```bash
npm run dev
```
